## TextMate bundle for Kohana v2.3.4 ##

## Installation ##

In the terminal:

$ cd ~/Library/Application Support/TextMate/Bundles  

$ git clone git://github.com/alexsancho/Kohana-2.3.4.tmbundle.git Kohana-2.3.4.tmbundle
